30 Second Honey Run


Fall 2020



HOW TO: WASD keys for movement.

DESCRIPTION: This game was a submission to the VimJam: Collectables game jam. I made it in one week. The premise is that you are a bear whose honey has been stolen. You have 30 seconds to complete two levels. Each level consists of parkour, collecting the honey, and parkouring back. This game was inspired by a game on Coolmathgames called "60 SECOND BURGER RUN".

TOOLS: Unity C#, Piskel
