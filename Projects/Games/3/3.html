1 Button Spy


Spring 2021



HOW TO: You are given a prompt in the top right, and a morse code key in the bottom left. Left-Click the telegraph to input a dot, Right-Click the telegraph to input a dash,

DESCRIPTION: This game was a submission to the 1-Bit Jam. I made it in two days. The premise is that you are a spy sending secret morse code messages.

TOOLS: Unity C#
