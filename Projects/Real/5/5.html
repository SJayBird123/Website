Winter & Summer 2022

PURPOSE: To calculate/generate various important metrics for First Robotics Competition scouting

HOW TO: Run the jar file. Select the year, country, and event you want to scout for. The app generates an xlsx file, and you choose where to save it. Open the file, and it will have a summary sheet with data for every team and team-specific qualification match breakdowns in subsequent sheets.

DESCRIPTION: I made this during the 2022 FRC season. At the time, no websites or programs that could conveniently display scouting data existed. The closest alternative was a website that took several minutes to load similar data, making it very difficult to use at competitions. This app simplifies the process, enabling users to run the program locally. During the 2022 Offseason, I further improved the program. The app starts by prompting the user for a year. All events from the selected year are parsed for every country that hosted an event. The user is prompted to pick a country. The user is prompted to pick an event from all events in the selected country and year. All matches from the event are stored and parsed for team names. The matches are broken down for all relevant data and stored in robot, alliance, and match data formats. Scoring data is used to determine the various OPRs and averages, data that is used as a predictive/advisory metric. That data is organized in a summary sheet by team numbers. There are individual team sheets that display individual match data. This is stored to a file in a location selected in a prompt by the user. A rerun prompt is shown to ask whether or not the user wants to rerun the program to choose another competition.
